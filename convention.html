camel case 為主, snake case為輔

名稱中使用英文字元, 不使用中文

同名原則, 例如: 在不同的table的相同意涵的欄位, 使用相同的欄位名稱

名稱中不使用空白

使用可以表達內容意涵的最短文字作為名稱

FileMaker 的名稱不區分大小寫, 在書寫時, 仍維持表達意涵的大小寫

如需要用縮寫, 取單字的第一個字元大寫表示

在 FileMaker 的 layout name, script name 允許使用以數字開頭的名稱, 應禁止使用這種寫法

在 FileMaker 的 script name 允許重複的名稱, 應禁止這種狀況

在 html, css, javascript 中的引號, 優先使用單引號

layout css style 名稱的命名規則

依循 css property 的命名習慣

格式: type_hv_background_foreground_size_serial_obj

example: eb_lc_trans_white_14_0_firstField

type 表示 layout object 的類別
  • te: text
  • eb: edit box, 包含 scrolling edit box, concealed edit box
  • dd: drop down list
  • pm: popup menu
  • cs: checkbox set
  • rb: radio button set
  • ca: drop down calendar
  • pa: part
  • bu: button
  • bb: button bar
  • po: pop over
  • wv: web viewer
  • tc: tab control
  • sl: slide
  • po: portal
  • ch: chart
  • li: line
  • re: rectangle
  • rr: rounded rectangle
  • ov: oval
hv 表示水平與垂直對齊
  • 水平方向: l: left, r: right, m: middle, f: fully justified
  • 垂直方向: t: top, c: center, b: bottom

background 背景色

foreground 前景色

size 字體大小

serial 序號

obj 標示用在哪裡或目的

table occurence 命名

left table«id»right table : relation on id of left table

left table«x»right table : Cartesian product

left table«1»right table : always connect relation

left table«比較»right table : other connection situation

«key_lt»less than

«key_le»less than or equal

«key_gt»greater than

«key_ge»greater than or equal

«key_ne»not equal

field name 命名

具有特殊意涵的 field 的結尾

field name 的結尾_Z: global field

field name 的結尾_U: unstored calculation field

field name 的結尾_S: summary field

field name 的結尾_V: value list

field name 的結尾_J: json

具有特殊意涵的 field 的附加結尾

field name: _html>表示是內容為 hmtl框架, 不含 mime 宣告

field name: _body>表示是內容為 hmtl的 body 內容, 不含 open tag 和 close tag

field name: _css>表示是內容為 css 的內容, 不含 open tag 和 close tag

field name: _js>表示是內容為 javascript 的內容, 不含 open tag 和 close tag

例如: menuTree_Z_html 表示是內容為 顯示 menuTree 內容的 html 框架, 是要被來源資料做置換

例如: menuTree_Z_css 表示是內容為 顯示 menuTree 內容的 css 的內容

custom function 的類別

cf_: 泛用型的 custom function

co_: 是常數的 custom function

mk_: file 或 table 專用的 custom function

_src: source

_dst : destination

_len : length

_ini or _start : initial state, starting point

_end : end point

_i : iteration, increment

_e : event parameter

_ts : timestamp

_if : boolean data, 如果未指定為 true 則視為 false(非真即假)

_is : boolean data, 如果未指定為 false 則視為 true(非假即真)

_ifCase : 是否區分大小寫, boolean data, 如果未指定為 true 則視為 false(非真即假)

_isCase : 是否區分大小寫, boolean data, 如果未指定為 false 則視為 true(非假即真)

_j or _json : json data

_key: key, also name of json data

_vlu: value, also value of json data

_mi : minimum

_mx : maximum

_rS : row separator

_cS : column separator

_lDiv : leading divider ( delimiter )

_tDiv : trailing divider ( delimiter )

_cr : color

_sz : size

_fo : font

_st : style

_toRep : to replace,是以事件發生時的狀態而定

_beRep : be replaced, 是以事件發生時的狀態而定
